# Guide Administrateur ONG

## Bienvenue dans votre plateforme de collecte de données !

Ce guide vous explique comment utiliser l'application pour gérer vos projets de collecte de données.

## 1. Connexion

### Première connexion
1. Ouvrez votre navigateur web
2. Allez sur l'URL fournie par votre administrateur
3. Entrez votre **code d'accès** (exemple : ADM-123)
4. Cliquez sur "Se connecter"

### Rôles dans l'organisation
- **Admin** : Contrôle complet de l'organisation
- **Project Manager** : Gestion des projets et équipes
- **Enumerator** : Collecte de données sur le terrain

## 2. Gestion des Projets

### Créer un nouveau projet
1. Cliquez sur "Nouveau Projet" dans le menu principal
2. Choisissez votre méthode d'import :
   - **Import XLSForm** : Uploadez un fichier Excel (.xlsx)
   - **Création assistée** : Utilisez l'IA pour créer un formulaire
   - **Modèle** : Choisissez parmi les modèles prédéfinis

### Configuration du projet
- **Nom du projet** : Donnez un nom descriptif
- **Description** : Expliquez l'objectif du projet
- **Paramètres** :
  - Collecte anonyme ou nominative
  - Validation automatique
  - Notifications par email

### Gestion des formulaires
- **Aperçu** : Visualisez le formulaire avant publication
- **Test** : Testez le formulaire en mode brouillon
- **Publication** : Rendez le formulaire disponible aux enquêteurs

## 3. Gestion de l'Équipe

### Ajouter des enquêteurs
1. Allez dans "Équipe" ou "Utilisateurs"
2. Cliquez sur "Ajouter un utilisateur"
3. Entrez le nom de l'enquêteur
4. Sélectionnez le rôle "enumerator"
5. Le système génère automatiquement un code d'accès

### Distribuer les codes d'accès
- **Format** : NOM-123 (ex: MAR-456)
- **Distribution** : Par email, WhatsApp, ou en personne
- **Sécurité** : Chaque code est unique et personnel

### Gérer les permissions
- **Activer/Désactiver** : Contrôlez l'accès des utilisateurs
- **Changer les rôles** : Promouvez un enquêteur en manager
- **Réinitialiser les codes** : En cas de perte ou compromission

## 4. Collecte de Données

### Suivre les soumissions
1. Allez dans "Soumissions" ou "Données"
2. Visualisez toutes les soumissions en temps réel
3. Filtrez par :
   - Projet
   - Période
   - Statut (Brouillon, Approuvé, Rejeté)

### Valider les données
- **Approuver** : Données correctes et complètes
- **Rejeter** : Demander des corrections à l'enquêteur
- **Commentaires** : Ajoutez des notes pour l'enquêteur

### Export des données
1. Sélectionnez les soumissions à exporter
2. Choisissez le format :
   - **Excel** (.xlsx) : Pour analyse dans Excel
   - **CSV** : Pour import dans d'autres outils
   - **PDF** : Rapport formaté
3. Téléchargez le fichier

## 5. Analytics et Rapports

### Tableaux de bord
- **Vue d'ensemble** : Nombre de soumissions, taux de completion
- **Cartes** : Visualisation géographique des données
- **Graphiques** : Tendances temporelles

### Analyse IA (Gemini)
1. Cliquez sur "Analyse IA"
2. Posez vos questions en langage naturel :
   - "Quel est le taux de réponse moyen ?"
   - "Montre-moi les tendances par région"
   - "Identifie les valeurs aberrantes"
3. Recevez des analyses instantanées

### Rapports automatiques
- **Quotidien** : Résumé des nouvelles soumissions
- **Hebdomadaire** : Statistiques détaillées
- **Mensuel** : Rapport complet avec graphiques

## Questions Fréquentes

**Q : Comment créer un formulaire ?**
R : Utilisez l'import XLSForm ou la création assistée par IA.

**Q : Un enquêteur a perdu son code ?**
R : Générez un nouveau code dans la section "Équipe".

**Q : Comment exporter mes données ?**
R : Allez dans "Soumissions" > "Exporter" et choisissez votre format.

**Q : L'application fonctionne-t-elle offline ?**
R : Oui, les enquêteurs peuvent collecter des données sans connexion internet.

**Q : Comment contacter le support ?**
R : Utilisez le chat en ligne ou envoyez un email à support@multi-ong-app.com.
